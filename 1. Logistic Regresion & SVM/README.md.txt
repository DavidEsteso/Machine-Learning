# Machine Learning Classification Models Analysis

This project implements and analyzes different machine learning classification models, comparing their performance and characteristics. The implementation includes Logistic Regression, Support Vector Machines (SVM), and enhanced models with squared features.

## Project Overview

The project analyzes classification models through three main components:

1. **Logistic Regression Analysis**
   - Dataset visualization and feature analysis
   - Model training and parameter interpretation
   - Decision boundary visualization
   - Prediction analysis

2. **Support Vector Machine (SVM) Implementation**
   - Analysis of regularization parameter C's impact
   - Training models with different C values
   - Comparative analysis of model performance
   - Visual representation of decision boundaries

3. **Advanced Feature Engineering**
   - Implementation of squared features
   - Performance comparison with linear models
   - Enhanced decision boundary analysis
   - Baseline predictor evaluation

## Project Structure

```
├── exercise_a.py      # Logistic Regression implementation
├── exercise_b.py      # SVM implementation
├── exercise_c.py      # Squared features analysis
├── main.py           # Main execution script
```

## How to Run

The project is structured to run all analyses through the main script:

```bash
python main.py
```

This will:
1. Load and visualize the initial dataset
2. Train and evaluate the logistic regression model
3. Perform SVM analysis with different C values
4. Generate and analyze models with squared features
5. Create visualizations and performance reports

## Output

The program generates:
- Visualization plots for each model
- Performance metrics and analysis reports
- Model parameters and coefficients
- Decision boundary visualizations
